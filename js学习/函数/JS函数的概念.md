#### JS函数的概念

```
编写js中函数的概念：
                函数：当它被调用的时执行的可重复使用的代码块

            函数的定义：
                无参函数的声明：
                function 函数名(){
                    函数体;
                }
                函数体内，外面怎么写js代码，这里也如何去写js代码
                可以封装任意多条语句，而且可以在任意地方，任何时候调用
                封装函数的时候。函数名=标识符 必须见名思意。当前函数的函数名必须体现出函数的功能

                函数分类：
                    内置函数（系统函数、官方函数）
                    alert() parseInt() document.write()等等

                    用户自定义函数
                    用户根据实际需求，自己封装一个函数

            函数的概念
            函数就是把完成特定功能的一段代码[抽象出来]，使之成为程序中的一个[独立实体]，起个名字（函数名）。
            可以在用一个程序或者其他程序中多次重复使用（通过函数名调用）。

            函数的作用
                1、使程序变得更加简短而清晰
                2、有利于程序的维护
                3、可以提高程序开发的效率
                4、提高了代码的重用性（复用性）
```

```javascript
//需要调用
//执行程序 程序被cpu所执行
function print() {
    for (let i = 0;i<10;i++){
        document.write('hello world');
    }
}
```
