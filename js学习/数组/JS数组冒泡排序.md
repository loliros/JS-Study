#### JS数组冒泡排序

```
原理：前后两数两两进行比较，如果符合交换的条件，就交换位置，直到所有得数据排序完成，才结束比较

第一轮
1 3 5 7 4 2
1 3 5 4 7 2
1 3 5 4 2 7

第二轮
1 3 4 5 2 7
1 3 4 2 5 7

第三轮
1 3 2 4 5 7

第四轮
1 2 3 4 5 7
```

```javascript
//按大小排序 从小到大
let arr = [1,5,3,7,4,2];
let num = 0;
for (let i=arr.length-1;i>=0;i--){//第i轮比较
    for (let j=0;j<=i;j++){//第i轮比较中的j和j+1进行比较
        if (arr[j]>arr[j+1]){
            num = arr[j];
            arr[j] = arr[j+1];
            arr[j+1] = num;
        }
    }
}
document.write(arr);//1,2,3,4,5,7
```
