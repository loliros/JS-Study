#### JS数组引用详解

```
声明在当前函数中的变量和当前函数的形参，作用域都限制在该函数内。
当函数调用结束后，这些变量和形参会被释放

数组 复合数据类型/引用数据类型。
所有的复合数据类型都不直接存储在我们的函数中，存储在堆段。
所有的复合数据类型，存储的都是地址，所以这种类型的数据，叫做引用数据类型
函数运行的内存空间是预先分配好的，一旦被分配好空间，就不能进行改变了。
在堆段，我们可以想要使用内存，就随时分配多少内存。

函数中的内存是事先分配好的，无法随便进行修改
```

```javascript
let arr = [10,1];
function show(arr) {
    arr.push(20,30);
    alert(arr);
}
alert(arr);//10,1
show(arr); //10,1,20,30
alert(arr);//10,1,20,30
```
