JS认识日期对象

```
在js中日期对象是复合数据类型
日期对象的创建方式
let time = new Date();
参数：如果我们不传递参数，默认获取的就是当前系统时间
我们所用的事件都是根据格林尼治时间

生成日期的参数
参数类型：
("2015-08-22")
(2016,04,22,14,25);
("2015/08/22")

1秒 = 1000毫秒
参数可以毫秒计算，计算从1970年开始+毫秒数，计算出这个时间
```

```javascript
let time = new Date();
let time1 = new Date(1000);//1970-1-1 因为1970年时unit的诞生日
alert(time);//Tue Jan 07 2020 12:38:48 GMT+0800 (中国标准时间)
```
